# Andrew Szigethy Portfolio

This repository maintains a professional portfolio website for Andrew Szigethy.

## What is included

- Home, About, Experience, Projects, Leadership, Writing, and Contact pages
- Blog posts stored as Markdown files in `_posts`
- Public portfolio content managed through source files
- A small public blog with file-based article content

## Site focus

The site presents Andrew's engineering leadership, software architecture, platform thinking,
delivery systems work, and writing in a clean static format.

## Notes

Private working notes and setup references, when needed, are kept outside the public root.
